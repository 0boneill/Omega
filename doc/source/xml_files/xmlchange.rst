.. _xmlchange:

#################
xmlchange script
#################

**xmlchange** is a python script in CIMEROOT/scripts/Tools that
can be used to modify XML variables stored in the CASEROOT/env_``*``.xml
files. It is copied into the CASEROOT as part of create_newcase, create_test,
and create_clone.
