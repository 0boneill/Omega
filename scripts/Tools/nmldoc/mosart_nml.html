logical
Valid Values
['.true.', '.false']
Default Value
value is .true. for: {'CLM_ACCELERATED_SPINUP': 'off', 'MOSART_MODE': 'ACTIVE'}
value is .false. for: {'CLM_ACCELERATED_SPINUP': 'on'}
value is .false. for: {'MOSART_MODE': 'NULL'}
