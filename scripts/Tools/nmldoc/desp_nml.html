Component Model Namelist Definitions for DESP

CESM Version: CESM2.0
HTML created on: 2017-03-30

This page contains the complete list of DESP namelist variables available.
They are grouped by categories designed to aid browsing.
Clicking on the name of a variable will display additional descriptive information.
Click on the "Show Details" button and then cntl+F key to search for specific strings in this file.

Category: desp

desp_mode The mode of operation for the DESP model. NOCHANGE: report status of model ...
Group: desp_nml

Description
The mode of operation for the DESP model. NOCHANGE: report status of model 'pause' but make no changes DATATEST: make a roundoff change to the restart files of the components in the 'PAUSE_COMPONENT_LIST' XML variable. Default: NOCHANGE
Type
char
Valid Values
['NOCHANGE', 'DATATEST']
Default Value
value: $DESP_MODE
XML Group Name
desp_nml
